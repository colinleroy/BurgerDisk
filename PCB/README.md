In this directory are the Kicad sources for the different PCBs.
Variations exist:
- *-Angled: For a right-angle, PCB-type DB19F connector (otherwise, panel-mount (solder cups) connector required)
- *-THT: Through-hold PCB. Will require an Arduino and SD module
- *-SMD: SMD PCB with the Atmega328p integrated, and the microSD where applicable

PCBs families:
- BurgerDisk: The simpler PCB, with more enclosures options (Full or micro SD)
- Mini: Everything integrated, for the Mini enclosure
- Domino: Everything integrated, as small as possible.
- (micro)SD-module: the card readers for the BurgerDisk
